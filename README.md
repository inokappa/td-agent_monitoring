# READE ME

## How to use

 1. git clone
 1. apt-get install libsqlite3-dev(for debian and ubuntu) 
 1. gem install bundler 
 1. bundle install
 1. add td-agent host ip to `list.txt`
 1. bundle exec rackup config.ru
